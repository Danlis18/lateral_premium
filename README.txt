#LATERAL WebSite


Features:
* Responsive design (desktop / tablet / mobile)
* Dark / Light theme with localStorage
* Instant media switching (images + video)
* Hero video (WebM + MP4 fallback)
* Tabs system for channels
* Expandable content cards
* Animated statistics & charts
* Contact form + resume modal (UI ready)
* FAQ section (SEO optimized)


📁 Project Structure
│
├── index.html          # Main HTML file (structure + SEO)
├── styles.css          # Styles and responsive design
├── script.js           # Frontend logic (theme, UI interactions)
│
├── assets/             # Static assets (images, videos, icons)
│
├── robots.txt          # Search engine crawling rules
├── sitemap.xml         # Sitemap for SEO indexing
│
├── server.js           # Basic Node.js server
├── package.json        # Project configuration and dependencies
├── package-lock.json   # Dependency lock file
├── node_modules/       # Installed npm packages
│
├── README.txt          # Project documentation
└── start.bat           # Local server start script (Windows)
